
Программа "Печатники"


1. Запуск  и загрузка  фильма

Запускаемый файл - "Pechatniki.exe" .

В параметрах программы  можно задавать название фильма. Например,
	Pechatniki test10.smil 

После того, как программа  запуcтилась фильм можно выбрать через меню:
 (меню "File" далее "Open").

Все фильмы имеют расширение *.smil.

После загрузки фильма  на экран  выводится первый кадр фильма.


2.  Просмотр

Чтобы начать просмотр фильма от текущего слайда, нужно сделать одно из трех:
	1. Нажать Ctrl-P;
	2. В меню выбрать "File" далее "Play";
	3. Нажать на кнопку со стрелкой вправо.

Чтобы остановить просмотр фильма, нужно сделать одно из трех:
	1. Нажать Ctrl-Q;
	2. В меню выбрать "File", далее "Stop";
	3. Нажать на кнопку со знаком "Stop".

Чтобы приостановить  просмотр фильма (пауза), нужно сделать одно из трех:
	1. Нажать пробел;
	2. В меню выбрать "File", далее "Pause";
	3. Нажать на кнопку с буквой "П".


Чтобы переместиться на один слайд вперед, нужно сделать одно из двух:
	1. Нажать стрелку вправо;
	2. В меню выбрать "Navigation", далее "Next Slide";

Чтобы переместиться на один слайд назад, нужно сделать одно из двух:
	1. Нажать стрелку влево;
	2. В меню выбрать "Navigation", далее "Previous Slide";

Чтобы перейти к слайду по заданному номеру, нужно сделать одно из двух:
	1. Нажать стрелку Ctrl-g;
	2. В меню выбрать "Navigation", далее "Goto Slide";


Чтобы показать слайд на полный экран, нужно нажать F9,
Чтобы показать слайд не на полный экран, нужно снова  нажать F9 или ESC.
	

3. Редактирование

Чтобы  вывести окно редактирования, нужно нажать F2, чтобы убрать его, нужно
снова нажать F2.
Когда видно окно редактирования, по-прежнему можно работать в основном окне. В обеих
окнах можно работать одновременно. В окне редактирования слева находится 
перечень слайдов фильма. Справа - две группы  кнопок: 
	1. Группа кнопок Periods (позволяет редактировать перечень слайдов).
	2. Группа кнопок Audio (позволяет привязывать слайд к звуку).
Перечень слайдов фильма состоит из двух столбцов (название файла со слайдом и 
длительность показа слайда). Текущий слайд в перечне выделен цветом. 
При смещении курсора  в перечне в основном окне автоматически показывается 
текущий слайд.  Кроме этого, справа снизу от  текущего слайда показывается 
следующий слайд в белой рамке.
	Группа "Periods" состоит из следующих кнопок:
	1. Edit (изменение название файла или длительности текущего слайда);
	2. Add (добавление нового слайда в конец перечня);
	3. Insert (вставка нового слайда перед текущим);
	3. Delete (удаление текущего слайда);
	Название файла должно относиться к реально существующему файлу, который лежит в том
же каталоге, что и основной файл smil. Если файл не найден, изменения не происходят.

	Группа "Audio" состоит из следующих кнопок:
	1. "Play" - проигрывание только звука фильма (при этом кадры автоматически не меняются )
	2. "Stop" - остановка звука;
	3. "Show next" - ручная смена кадра.
	Предполагается следующий сценарий работы с этой группой. Пользователь  ставит 
курсор на некоторый кадр и нажимает "Play". Звук начинает идти от текущего слайда.
Когда пользователь нажимает "Show next", кадр меняется, при этом в перечень слайдов 
записывается продолжительность показа этого слайда. 
	Таким образом, когда  мы имеем  отсканированные кадры, которые записаны в smil файл,
чтобы приписать кадры к звуку, нам нужно только один раз просмотреть фильм и вовремя
нажимать на кнопку след. кадр ("Show next"). 
	Любые изменения в перечне слайдов нужно сохранить на диск.

	В основном окне, чтобы  сохранить перечень, нужно сделать одно из трех:
	1. Нажать Ctrl-S;
	2. В меню выбрать "File" далее "Save";
	3. Нажать на кнопку с дискеткой.
