Шаг 2: Как загрузить файлы на виртуальный сервер

Чтобы готовые страницы веб-сайта были доступны в интернете, их нужно загрузить на виртуальный сервер. Для передачи файлов между двумя компьютерами, подключенными к сети интернет (в данном случае - для передачи файлов с Вашего компьютера на сервер, на котором будет размещаться веб-сайт), проще всего использовать протокол FTP (протокол передачи файлов).

Процесс загрузки файлов удобно осуществлять с помощью специальных программ, называемых FTP-клиентами. При использовании таких программами Вам не нужно знать принципы работы FTP-протокола. Их удобный интерфейс позволяет обмениваться файлами с сервером так же легко, как копировать файлы с одного диска компьютера на другой. Большинство программ, поддерживающих обмен файлами по протоколу FTP, имеет интерфейс, похожий на обычные файловые менеджеры для Вашей операционной системы (как правило, это Windows). Поэтому все, что нужно сделать - это настроить параметры подключения к виртуальному серверу и просто скопировать необходимые файлы с Вашего жесткого диска в директорию на сервере.

Вот некоторые популярные FTP-клиенты:

  • Сute FTP
  • FAR Manager
  • Windows Commander

При работе с FTP-клиентом необходимо знать параметры для подключения к серверу:

  • имя узла (хоста), к которому должен подключаться FTP-клиент - home.sokirko.info.
  • имя пользователя (логин) и пароль - выдаются Вам при регистрации контракта.

Эти параметры необходимо использовать при настройке FTP-клиента для обновления информации на Вашем сайте. О том, как настроить Вашу программу (FTP-клиент), можно прочитать в документации к ней.

Подключившись к виртуальному серверу с этими параметрами, Вы автоматически попадете в Вашу домашнюю директорию. Здесь расположена директория www, в которую нужно скопировать все файлы, относящиеся к сайту. Будьте внимательны - посетители сайта смогут увидеть или загрузить только те страницы и файлы, которые находятся в директории www Вашего виртуального сервера.

После того, как Вы загрузили все необходимые файлы на виртуальный сервер, сайт готов к приему посетителей. Зайдите по адресу http://www.sokirko.info - и Вы увидите свой сайт в сети интернет.
